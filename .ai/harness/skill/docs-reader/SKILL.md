# 文档读取技能

此技能用于读取各种格式的文件内容，返回原始内容供后续处理使用。

## 支持的文件类型

### 源代码文件
- 编程语言：.py, .js, .ts, .jsx, .tsx, .java, .go, .rs, .cpp, .c, .h, .cs, .rb, .php, .swift, .kt, .vue, .svelte
- 脚本语言：.sh, .bash, .zsh, .ps1, .bat

### 配置文件
- 结构化配置：.json, .yaml, .yml, .toml, .xml, .ini, .conf, .cfg
- 环境配置：.env, .properties

### 文档格式
- 文本文档：.md, .txt, .rst, .adoc
- 网页格式：.html, .htm

### 架构图和图片
- 图表格式：.drawio, .puml, .plantuml
- 图片格式：.svg, .png, .jpg, .jpeg, .gif, .bmp, .webp

### 数据文件
- 数据格式：.csv, .tsv
- 数据库：.sql, .db, .sqlite

## 使用方法

### 基本读取
直接使用Read工具读取文件内容：

```
读取文件：/path/to/file.py
```

### 批量读取
读取多个文件或目录：

```
读取整个目录：/path/to/directory/
读取多个文件：file1.py, file2.js, config.json
```

### 特殊处理
对于特定格式，可能需要额外处理：

1. **二进制文件**：显示文件大小和类型信息，不尝试读取内容
2. **大型文件**：如果超过100KB，提示用户可以分批读取
3. **编码问题**：尝试多种编码（UTF-8, GBK, ISO-8859-1）

## 输出格式

返回文件内容，包含以下元信息：
- 文件路径
- 文件大小
- 文件类型
- 编码格式
- 最后修改时间

## 错误处理

- 文件不存在：返回错误信息并建议检查路径
- 权限不足：提示需要相应权限
- 文件损坏：报告损坏情况
- 格式不支持：建议支持的格式列表
