# Drawio图表生成技能

此技能用于将结构化数据或文本描述转换为Drawio格式的图表文件。

## 支持的图表类型

### 1. 流程图
- 顺序流程图
- 条件分支流程图
- 并行流程图
- 循环流程图

### 2. 架构图
- 系统架构图
- 网络拓扑图
- 微服务架构图
- 分层架构图

### 3. 组织结构图
- 树状组织图
- 层级结构图
- 矩阵结构图

### 4. 类图/UML
- 类图
- 时序图
- 状态图
- 活动图

## 输入格式

### 结构化数据格式
```json
{
  "type": "architecture",
  "root": "系统名称",
  "categories": [
    {
      "name": "分类1",
      "nodes": ["节点1", "节点2"]
    }
  ]
}
```

### 文本描述格式
```
根节点: 系统
  分类: 前端
    节点: Web端
    节点: 移动端
  分类: 后端
    节点: API服务
    节点: 数据库
```

## 输出格式

生成Drawio格式的XML文件，可直接在Drawio编辑器中打开。

### 布局规则
- **根节点**：位于顶部居中或左侧
- **一级分类**：水平排列在根节点下方
- **功能节点**：根据兄弟节点数量采用水平分支布局
- **节点间距**：动态计算，避免重叠
- **连接线**：使用曲线边，避免直线交叉

### 节点样式
- **根节点**：圆角矩形，深色填充
- **分类节点**：圆角矩形，中色填充
- **功能节点**：圆角矩形，浅色填充
- **连接线**：贝塞尔曲线，箭头指向

## 使用方法

### 基本用法
```
生成架构图：系统包含前端和后端两部分
```

### 指定类型
```
生成流程图：用户登录流程
```

### 复杂图表
```
根据以下结构生成架构图：
{
  "root": "电商系统",
  "categories": [...]
}
```

## 输出要求

- 输出为.drawio.svg或.drawio文件
- 符合Drawio XML规范
- 布局紧凑，无重叠
- 线条清晰，无交叉
- 支持中文显示

## 错误处理

- 输入格式错误：提示正确的输入格式
- 数据不完整：提示缺失的必需字段
- 布局冲突：自动调整节点位置
