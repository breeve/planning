# 文档解析技能

此技能用于解析各种文档格式，提取结构化信息和关键内容。

## 功能特点

### 结构提取
- **标题层级**：识别H1-H6标题，建立文档大纲
- **列表识别**：解析有序列表和无序列表
- **表格解析**：提取表格数据，转换为结构化格式
- **代码块识别**：识别代码块，提取语言类型

### 内容分析
- **段落提取**：提取正文段落，过滤无关内容
- **链接提取**：提取所有链接及其上下文
- **图片提取**：识别图片引用和路径
- **元数据提取**：提取创建时间、作者、标签等信息

### 格式识别
- **Markdown解析**：解析标准Markdown语法
- **HTML解析**：转换HTML为Markdown等价表示
- **JSON解析**：提取JSON结构化数据
- **XML解析**：解析XML标签结构

## 支持的解析模式

### 1. 源代码解析
提取：
- 函数和类定义
- 导入依赖
- 注释和文档字符串
- 类型注解

### 2. 配置文件解析
提取：
- 配置项和值
- 嵌套结构
- 注释说明

### 3. 文档解析
提取：
- 章节结构
- 关键段落
- 代码示例
- 引用内容

### 4. 架构图解析
提取：
- 节点信息
- 连接关系
- 分组结构
- 样式属性

## 使用方法

### 基本解析
```
解析文档结构：/path/to/document.md
```

### 提取特定信息
```
提取所有代码块
提取所有表格
提取所有链接
```

### 深度分析
```
分析文档结构
提取函数列表
获取API端点
```

## 输出格式

返回结构化的解析结果：
- 结构树：文档的层级结构
- 元素列表：所有关键元素及其位置
- 统计数据：字符数、段落数、代码块数等

## 错误处理

- 格式无法识别：返回原始内容并标注格式
- 解析失败：报告失败原因和位置
- 部分解析：返回已解析部分和剩余部分
